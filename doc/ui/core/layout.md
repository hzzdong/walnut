# 控件概述

Layout 控件假想你会按照下面两种格式布局一个界面


```
纵向布局 (vertical)
+-------------------------------------------+
|                      l0                   |
+-------------------------------------------+
|                      l1                   |
+-------------------------------------------+
|                      l2                   |
+-------------------------------------------+


横向布局 (horizontal)
+-------------+--------------+--------------+
|             |              |              |
|             |              |              |
|     l0      |      l1      |      l2      |
|             |              |              |
|             |              |              |
+-------------+--------------+--------------+
```

* 无论那种布局，布局内部都会嵌入一个子 UI

# 如何创建实例

```
new UILayout({
    $pel    : $(document.body),
    mode    : "vertical" | "horizontal",
    display : [80, "20%", "*"] // 每个区域的显示尺寸
    setup   : [{..}, {..}]       // 每个对象都是控件区域的配置
}).render();
```

# 控件区域的配置

```
{
    uiType  : "name/to/ui",
    uiConf  : {
        // 这个就是对应 UI 的配置
        // 只是会无视 $pel 和 fitparent 等通用配置参数
    }
}
```
